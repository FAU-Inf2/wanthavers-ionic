# wanthavers-ionic

## Project Setup
- Install node.js https://nodejs.org/en/
- Install ionic http://ionicframework.com/getting-started/
- Clone this repository
- ```cd wanthavers-ionic```
- ```npm install```

## Developing
To start developing in a browser (use Google Chrome!):
- ```ionic serve```
- Enter dev mode by pressing ```ctrl-shift-i```
